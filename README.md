# Instruções de Instalação e Execução

Clone este repositório:
```bash
git clone https://github.com/VitorArgeri/Atividade-Avaliativa-Front
```

Instale o Node_Modules
```bash
npm install
```

Execute o servidor
```bash
npm run dev
```
